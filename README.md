# Games
This repository contains the games I've built while learning game development . Each project reflects my progress and understanding . Feel free to explore...
# 1 .  🎮 Rooftop Rush

A 3D endless runner game where a guy runs, walks, and jumps across rooftops and floors. 
Built from scratch using **Unity** as my very first game development project!

## 🕹️ Gameplay Preview

👉 Watch the gameplay here: https://youtu.be/cqDc0zBeF1M?si=igtnUiOb7q1_xKpX

## 🛠️ Built With

- Unity (3D)
- C#

## 🚀 About This Project

This is my **first ever game** built in Unity. 
The player controls a 3D character who runs, jumps, and navigates 
across houses and floors. It was an amazing learning experience diving 
into game development for the very first time!

> *Every big game started with a single line of code.* 💡
